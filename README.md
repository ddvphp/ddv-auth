ddv-auth
===================

Installation - 安装
------------

```bash
composer require ddvphp/ddv-auth
```

Usage - 使用
-----

### 1、`canonicalQuerySort`

```php

\DdvPhp\DdvAuth\Sign::canonicalQuerySort();

```

### 2、`getCanonicalHeaders`

```php

\DdvPhp\DdvAuth\Sign::getCanonicalHeaders();

```
